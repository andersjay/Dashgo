
## Dashgo

Dashboard application created using NextJS with Typescript.

by Rocketseat.




### Running Locally

Clone the project

```bash
  git clone https://github.com/andersjay/Dashgo.git
```

Enter the project directory

```bash
  cd dashgo
```

Install the dependencies

```bash
  yarn
  or
  npm install
```

Start the server

```bash
  yarn dev 
  or
  npm run dev
```


### Dependencies

- Next JS
- Typescript
- Mirage JS
- Axios
- Apex Charts
- React Query
- Yup 
- Chakra Ui
